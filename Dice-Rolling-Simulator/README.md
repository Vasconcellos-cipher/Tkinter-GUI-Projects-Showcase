# 🎲 Dice Rolling Simulator

A simple desktop application built with **Python** and **Tkinter** that simulates rolling a six-sided dice. Every button click generates a random number between 1 and 6, allowing users to practice event-driven programming and graphical user interface development.

This project was created as part of my Python GUI learning journey, focusing on Tkinter widgets, event handling, and Python fundamentals.
