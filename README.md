## 🛠️ Running a Project

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/tkinter-gui-projects.git
```

Navigate to the desired project:

```bash
cd tkinter-gui-projects/catalogo
```

Run the application:

```bash
python movie_and_series_catalog.py
```
